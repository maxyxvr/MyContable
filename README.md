# 📊 MyContable — Sistema Contable Web Premium

![Banner](https://img.shields.io/badge/Status-100%25_Funcional-emerald?style=for-the-badge&logo=github)
![Security](https://img.shields.io/badge/Seguridad-AES--256_Encrypted-blue?style=for-the-badge&logo=lock)
![UI](https://img.shields.io/badge/Design-Glassmorphism-purple?style=for-the-badge)

**MyContable** es una solución de contabilidad moderna, rápida y segura diseñada para pequeñas empresas y contadores independientes. Construida puramente con tecnologías web estándar (Vanilla JS), ofrece una experiencia de usuario fluida con una estética de vanguardia y cifrado de datos local.

---

## ✨ Características Principales

- **🛡️ Máxima Seguridad:** Todos los datos se cifran localmente usando el algoritmo **AES-256**. Tu información es ilegible para terceros.
- **🎨 Interfaz Premium:** Diseño basado en *Glassmorphism*, con fondos animados, tipografía elegante y una experiencia de usuario altamente intuitiva.
- **📦 15+ Módulos Integrados:**
  - Facturación Electrónica (Simulada)
  - Gestión de Clientes y Proveedores
  - Control de Gastos e Ingresos
  - Nómina Prestacional (Liquidaciones, Seguridad Social)
  - Conciliación Bancaria
  - Reportes Financieros en Tiempo Real
  - Control de Inventarios
- **🚀 Sin Backend:** Funciona completamente en el cliente aprovechando el `localStorage` del navegador. Ideal para uso privado y rápido.

---

## 🛠️ Tecnologías

- **Frontend:** HTML5, CSS3, Vanilla JavaScript.
- **Estilos:** Tailwind CSS (via CDN) con personalización de diseño.
- **Seguridad:** [CryptoJS](https://github.com/brix/crypto-js) para cifrado AES-256.
- **Iconos:** Google Material Symbols.
- **Fuentes:** Manrope & Inter (Google Fonts).
